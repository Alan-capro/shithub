# 😎🐙🪏 Shithub

> **Where bugs become features**

A tongue-in-cheek parody of the GitHub landing page – a single-page React app where bugs become features, contribution graphs turn into a brown swamp, and the merge button is actively afraid of you.

## ✨ Features

### 🎭 Parody GitHub-style Header
- Custom dark **"sludge"** color palette
- Fake search box (`Search for garbage...`)
- **Sign up** button that always reports: *"Database is currently on fire"*

### 🎯 Hero Section
- Tagline: **"Where bugs become features"**
- Call-to-action button: **"Start Shitting"** (purely cosmetic)

### 🟤 Contribution Swamp
A fake contribution heatmap rendered as a brown grid:
- Generated by `generateBrownContributionGraph`
- Simulates up to 16 "commits" per day
- Maps commit density to a brown color scale
- Shows tooltips like `X questionable commits` or, for heavy activity: **"Touch grass, please."**

### 💥 Trending Disasters
List of parody repositories such as:
- `Is-Even-As-A-Service` – "Parity checks monetized for absolutely no reason."
- `left-pad-but-worse` – "Now breaks both sides equally."
- `COBOL-for-Web3` – "Mainframe vibes, blockchain mistakes."

Each repo includes:
- `name`, `description`, `top_language`
- `stats` (`facepalms`, `spoons`, `regrets`)
- `last_update` (always something ominous like *"5 minutes before the server crashed"*)

### 🎲 Pull Request Simulator
- A **Merge** button that tries to escape your mouse
  - On hover, `calculateRunawayMergePosition` pushes the button away from the cursor
- 35% chance of triggering a **philosophical alert**:
  - e.g., *"If code is unreadable, is it still elegant?"*
  - When that happens, the merge is disabled
- On click:
  - If not "philosophically approved": **"Merge denied: existential checks still running."**
  - Otherwise, a random "Shithub status code" from `[418, 500, 502, 599]` is returned

## 🚀 Getting Started

### Prerequisites

- **Node.js** ≥ 18 (required by Vite 5 and some dependencies)
- **npm** (or another Node package manager)

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Then open the printed URL (by default `http://localhost:5173`) in your browser.

### Build

```bash
npm run build
```

This will:
- Run TypeScript project build (`tsc -b`)
- Run `vite build` to output a production bundle into the `dist` directory

### Preview Production Build

```bash
npm run preview
```

This serves the contents of `dist` via Vite's preview server.

## 🔧 Core Modules

### `ShithubLanding` (`src/components/ShithubLanding.tsx`)

Main landing page component that:
- Uses `parodyRepos` from `src/data/repos.ts`
- Uses `generateBrownContributionGraph` from `src/lib/contributions.ts`
- Uses `calculateRunawayMergePosition`, `maybeAskPhilosophicalQuestion`, `randomShithubStatusCode` from `src/lib/pullRequest.ts`
- Renders all main UI sections and wires up user interactions (alerts, hover events, merge simulation)

### Contributions Utilities (`src/lib/contributions.ts`)

**`generateBrownContributionGraph(days = 119): ContributionCell[]`**
- Validates `days > 0`, otherwise throws `ItsNotABugItsAFeatureError`
- Creates a list of `ContributionCell` objects:
  - `date`: ISO date string (YYYY-MM-DD)
  - `commits`: random 0–15
  - `color`: chosen from brown scale by density
  - `tooltip`: human-readable summary string

### Pull Request Utilities (`src/lib/pullRequest.ts`)

- **`maybeAskPhilosophicalQuestion(): string | null`**
  - 35% chance of returning a philosophical question from `PHILOSOPHY_BANK`

- **`calculateRunawayMergePosition(mouseX, mouseY, buttonX, buttonY)`**
  - Computes a new position pushed away from the cursor by a constant distance (46px)

- **`randomShithubStatusCode(): number`**
  - Returns one of `[418, 500, 502, 599]`

### Types (`src/types/index.ts`)

- **`Repo`** – shape of a parody repository item
- **`ContributionCell`** – shape of each contribution heatmap cell

### Errors (`src/lib/errors.ts`)

- **`ItsNotABugItsAFeatureError`** – custom error class for contribution graph validation

## 🎨 Customization

### Add More Parody Repos

Edit `src/data/repos.ts` (and optionally `public/parody-repos.json`) to add new entries following the `Repo` type:

```typescript
{
  name: string;
  description: string;
  top_language: string;
  stats: {
    facepalms: string;
    spoons: number;
    regrets: string;
  };
  last_update: string;
}
```

### Tweak the Contribution Swamp

Adjust the following in `src/lib/contributions.ts`:
- `BROWN_SCALE` – color palette array
- `tooltipFor()` – tooltip text generation
- `shadeFor()` – color mapping logic
- `days` argument in `generateBrownContributionGraph()` – number of days to display

### Change the Look & Feel

- Update `tailwind.config.cjs` to change or extend the `sludge` palette
- Add or modify utility classes in `src/index.css` as needed

## 📝 Project Description (中文)

**这个项目是一个 GitHub 登陆页恶搞版 "Shithub" 单页网站。**  
技术栈是 **React 18 + TypeScript + Vite 5 + Tailwind CSS**，前端纯静态，不依赖后端。

### 核心功能模块

- **入口与基础结构**
  - `index.html`：挂载点 `#root`，通过 `<script type="module" src="/src/main.tsx">` 引入前端
  - `src/main.tsx`：用 `ReactDOM.createRoot` 渲染 `App` 组件，并引入全局样式 `index.css`
  - `src/App.tsx`：非常薄的一层，只负责渲染 `ShithubLanding` 页面组件

- **主界面 `ShithubLanding`**
  - 文件：`src/components/ShithubLanding.tsx`
  - 顶部是一个 **仿 GitHub 顶栏**，使用自定义暗色 "sludge" 调色板
  - **Hero 区域**：Slogan "Where bugs become features"，按钮 "Start Shitting"
  - **Contribution Swamp**：使用 `generateBrownContributionGraph` 生成 119 天的伪造贡献热力图
  - **Trending Disasters**：使用 `parodyRepos` 渲染多个恶搞仓库
  - **Pull Request Simulator**：Merge 按钮会躲避鼠标，35% 概率触发哲学问题

- **构建与样式**
  - `package.json`：包含 `dev`、`build`、`preview` 脚本
  - `tailwind.config.cjs`：配置 Tailwind 内容路径和 "sludge" 颜色调色板
  - `postcss.config.cjs`：PostCSS 配置（Tailwind + Autoprefixer）
  - `tsconfig.*`：TypeScript 项目引用配置（app + node）
