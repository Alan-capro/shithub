# Shithub

A parody GitHub-like landing page built with React + TypeScript + Tailwind CSS.

## Features

- Muddy dark theme that looks vaguely familiar, then progressively wrong.
- Navigation with `Search for garbage...` and a self-aware `Sign up` alert.
- Hero section with `Where bugs become features` and `Start Shitting` CTA.
- Parody repository feed using `Facepalms`, `Spoons`, and `Regrets` stats.
- Brown-shaded contribution graph with the tooltip `Touch grass, please.` for overcommitters.
- Pull Request simulator with a merge button that runs away or asks philosophical questions.
- Funny custom error class: `ItsNotABugItsAFeatureError`.

## How to run

```bash
npm install
npm run dev
```

## How to break it

1. Spam-hover the merge button and chase it forever.
2. Refresh until contribution squares look like pure espresso mud.
3. Click `Sign up` during demos and claim infra is a feature.
4. Pretend status code `599` is a stable product signal.

## Fake status code policy

- `418`: I am a teapot.
- `599`: Server is tired.
